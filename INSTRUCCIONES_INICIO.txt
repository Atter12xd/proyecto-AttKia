❌ Las fotos no se ven:
   → Verifica nombres: atter.png y kiara.png
   → Deben estar en carpeta images/

   → Descarga de: nodejs.org (versión 16+)
   → Luego ejecuta: npm install

❌ Quiero cambiar colores/contenido:
   → Edita styles.css para colores
   → Edita index.html para contenido
